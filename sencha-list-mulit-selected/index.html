sencha list实现多选

  • Sencha
  • hybrid
  • sencha
  • listview
  • checkbox

dataview和listview中有配置字段“mode”，对于该字段的说明：

Modes of selection. Valid values are ‘SINGLE’, ‘SIMPLE’, and ‘MULTI’. Defaults to: ‘SINGLE’. Available since: 2.0.0

MULTI就是多选
此外还提供了各种‘%select%’相关的方法，在API中搜索即可看到。
